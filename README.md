                        MOVIE REVIEW APP

A responsive Movie Review Web Application built using React.js that allows users to search for movies, view movie details, filter results, and navigate through pages of results using the OMDB API.

This project demonstrates the use of React Hooks, API integration, filtering, and pagination to create a dynamic user interface.

=> Features

        🔎 Search Movies by title using the OMDB API

        🎞 Movie Cards displaying poster, title, and year

        📄 Movie Details Page with extended information

        📅 Filter Movies by Year

        🔢 Pagination Support to browse multiple pages of results

        *   Dynamic Data Fetching using API calls

        📱 Fully Responsive Design for mobile and desktop

        🎨 Styled using Tailwind CSS


🛠️ Technologies Used


        React.js

        React Router DOM

        JavaScript (ES6)

        Tailwind CSS

        OMDB API

        Vite
